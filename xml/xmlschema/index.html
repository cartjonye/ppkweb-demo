About XML Schema

An XML schema is a description of a type of XML document, typically expressed in terms of constraints on the structure and content of documents of that type, above and beyond the basic syntactical constraints imposed by XML itself. XSD (XML Schema Definition) is a World Wide Web Consortium (W3C) recommendation that specifies how to formally describe the elements in an Extensible Markup Language (XML) document.

JavaScript client side XML Schema Validation

The linter is based on the xmlint.js developed by kripken. You can implement this module by including xmlint.js in this page. Call validateXML(Module) function to validate the xml.
Courtesy of syssgx.github.io/xml.js 

          
//create an object
var Module = {
      xml: xmlData,
      schema: schemaData,
      arguments: ["--noout", "--schema", schemaFileName, xmlFileName]
};
  
//and call function
var xmllint = validateXML(Module);

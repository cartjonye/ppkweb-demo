About MathML

Mathematical Markup Language (MathML) is a mathematical markup language, an application of XML for describing mathematical notations and capturing both its structure and content. MathML is a low-level specification for mathematical and scientific content on the Web and beyond.
MathML is capable to describe complex mathematical expressions and reconstruct it if necessary. But, due to the nature of XML itself, MathML is verbose in design and large in size.
MathML is platform independent, meaning that every software built to use MathML can make use of MathML data.

About MathJax

MathJax is a cross-browser JavaScript library that displays mathematical notation in web browsers, using MathML, LaTeX and ASCIIMathML markup. It's easy to install and implement. If your site is using a CMS like Wordpress, Moodle, or Drupal, there may be a plugin to add MathJax to your site already. Or if you write your own HTML, just put <script></script> snippet to load MathJax from CDN and you are ready to go.
